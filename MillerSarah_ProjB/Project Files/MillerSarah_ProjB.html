USER INSTRUCTIONS

Use arrow keys to navigate the scene, and Use WASD to pivot camera.
Use "p" to add petals to flower, and "o" to subtract petals.
Press the Space Bar to start or stop the star spinning. Click and drag to spin the star (whether it's moving or not).
Control the orthographic frustum parameters using a whole lot of buttons! 1&2 for Left, 3&4 for Right, 5&6 for Bottom, 7&8 for Top, R&T for Near, Y&U for Far.

Please use a browser that supports "canvas"
